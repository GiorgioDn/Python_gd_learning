This repository is created for learning purposes and serves as a workspace for projects within a Python and Machine Learning Academy.

## Description

It includes a variety of resources and exercises:
- **Python Fundamentals**: Basic syntax, conditions, and loops.
- **Object-Oriented Programming**: Concepts like encapsulation.
- **Data Science Libraries**: Practical examples using **NumPy** (broadcasting) and **Pandas** for data manipulation.
- **Databases**: SQL implementation using **SQLite3**.
- **Machine Learning**: Implementation of predictive models and exploratory data analysis (EDA).
